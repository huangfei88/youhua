# Windows Server 2022 手动极限精简优化教程（Azure 2核1G 专用）

> **适用环境：** Windows Server 2022，Azure 低配虚拟机（2核 1GB 内存）  
> **执行方式：** 以管理员身份打开 PowerShell，逐步粘贴执行下方每一条命令  
> **执行前提：** 右键"开始菜单" → "Windows PowerShell（管理员）"或"终端（管理员）"

---

## 第1步：切换系统语言为简体中文 + 时区 UTC+8

```powershell
Set-WinSystemLocale zh-CN
Set-WinUILanguageOverride -Language zh-CN
Set-Culture zh-CN
Set-WinUserLanguageList zh-CN -Force
Set-TimeZone -Id 'China Standard Time'
```

> ✅ 语言和时区设置需**重启后**完全生效。

## 第2步：关闭 Windows Defender

### 2.1 写入组策略注册表（彻底禁用）

```powershell
$defPath = 'HKLM:\SOFTWARE\Policies\Microsoft\Windows Defender'
New-Item -Path $defPath -Force | Out-Null
Set-ItemProperty -Path $defPath -Name 'DisableAntiSpyware'           -Value 1 -Type DWord
Set-ItemProperty -Path $defPath -Name 'DisableAntiVirus'             -Value 1 -Type DWord
Set-ItemProperty -Path $defPath -Name 'DisableRealtimeMonitoring'    -Value 1 -Type DWord
Set-ItemProperty -Path $defPath -Name 'DisableRoutinelyTakingAction' -Value 1 -Type DWord

New-Item -Path "$defPath\Real-Time Protection" -Force | Out-Null
Set-ItemProperty -Path "$defPath\Real-Time Protection" -Name 'DisableRealtimeMonitoring'   -Value 1 -Type DWord
Set-ItemProperty -Path "$defPath\Real-Time Protection" -Name 'DisableBehaviorMonitoring'   -Value 1 -Type DWord
Set-ItemProperty -Path "$defPath\Real-Time Protection" -Name 'DisableOnAccessProtection'   -Value 1 -Type DWord
Set-ItemProperty -Path "$defPath\Real-Time Protection" -Name 'DisableScanOnRealtimeEnable' -Value 1 -Type DWord

New-Item -Path "$defPath\Spynet" -Force | Out-Null
Set-ItemProperty -Path "$defPath\Spynet" -Name 'SpynetReporting'      -Value 0 -Type DWord
Set-ItemProperty -Path "$defPath\Spynet" -Name 'SubmitSamplesConsent' -Value 2 -Type DWord
```

### 2.2 停止并禁用 Defender 相关服务

```powershell
$defSvcs = @('WinDefend','WdNisSvc','Sense','SecurityHealthService','wscsvc','WdNisDrv')
foreach ($svc in $defSvcs) {
    $s = Get-Service -Name $svc -ErrorAction SilentlyContinue
    if ($s) {
        Stop-Service -Name $svc -Force -ErrorAction SilentlyContinue
        Set-Service  -Name $svc -StartupType Disabled -ErrorAction SilentlyContinue
        Write-Host "已禁用: $svc"
    } else {
        Write-Host "跳过(不存在): $svc"
    }
}
```

### 2.3 禁用 Defender 计划任务

```powershell
$defTasks = @(
    'Windows Defender Cache Maintenance',
    'Windows Defender Cleanup',
    'Windows Defender Scheduled Scan',
    'Windows Defender Verification'
)
foreach ($tn in $defTasks) {
    $dt = Get-ScheduledTask -TaskPath '\Microsoft\Windows\Windows Defender\' -TaskName $tn -ErrorAction SilentlyContinue
    if ($dt) {
        Disable-ScheduledTask -TaskPath '\Microsoft\Windows\Windows Defender\' -TaskName $tn -ErrorAction SilentlyContinue | Out-Null
        Write-Host "已禁用任务: $tn"
    } else {
        Write-Host "跳过(不存在): $tn"
    }
}
```

> ✅ Defender 需**重启后**完全关闭。

## 第4步：关闭不必要的 Windows 功能

```powershell
$features = @(
    'Internet-Explorer-Optional-amd64',
    'MediaPlayback',
    'WindowsMediaPlayer',
    'Printing-PrintToPDFServices-Features',
    'Printing-XPSServices-Features',
    'WorkFolders-Client',
    'FaxServicesClientPackage',
    'SMB1Protocol',
    'MicrosoftWindowsPowerShellV2Root'
)
foreach ($feat in $features) {
    $obj = Get-WindowsOptionalFeature -Online -FeatureName $feat -ErrorAction SilentlyContinue
    if ($null -eq $obj) {
        Write-Host "跳过(不存在): $feat"
    } elseif ($obj.State -eq 'Disabled') {
        Write-Host "跳过(已禁用): $feat"
    } else {
        Disable-WindowsOptionalFeature -Online -FeatureName $feat -NoRestart -ErrorAction SilentlyContinue | Out-Null
        Write-Host "已关闭: $feat"
    }
}
```

## 第6步：固定虚拟内存页面文件为 2048MB

```powershell
# 直接写注册表（比 WMI Win32_PageFileSetting 更可靠，Azure VM 上 WMI Put() 经常失败）
$mmKey = 'HKLM:\SYSTEM\CurrentControlSet\Control\Session Manager\Memory Management'
Set-ItemProperty -Path $mmKey -Name 'AutomaticManagedPagefile' -Value 0 -Type DWord
Set-ItemProperty -Path $mmKey -Name 'PagingFiles' -Value 'C:\pagefile.sys 2048 2048' -Type MultiString

Write-Host "页面文件已固定为 2048MB（重启后生效）"
```

## 第9步：注册表优化 + 网络（TCP）优化

### 9.1 注册表优化

```powershell
# 禁用 Cortana
New-Item -Path 'HKLM:\SOFTWARE\Policies\Microsoft\Windows\Windows Search' -Force | Out-Null
Set-ItemProperty -Path 'HKLM:\SOFTWARE\Policies\Microsoft\Windows\Windows Search' -Name 'AllowCortana' -Value 0 -Type DWord

# 禁用遥测数据收集
New-Item -Path 'HKLM:\SOFTWARE\Policies\Microsoft\Windows\DataCollection' -Force | Out-Null
Set-ItemProperty -Path 'HKLM:\SOFTWARE\Policies\Microsoft\Windows\DataCollection' -Name 'AllowTelemetry' -Value 0 -Type DWord

# 禁用错误报告
New-Item -Path 'HKLM:\SOFTWARE\Microsoft\Windows\Windows Error Reporting' -Force | Out-Null
Set-ItemProperty -Path 'HKLM:\SOFTWARE\Microsoft\Windows\Windows Error Reporting' -Name 'Disabled' -Value 1 -Type DWord

# 缩短服务停止等待时间为 2 秒（默认 20 秒）
Set-ItemProperty -Path 'HKLM:\SYSTEM\CurrentControlSet\Control' -Name 'WaitToKillServiceTimeout' -Value '2000'

# 禁用 IPv6（保留回环地址 ::1，避免 Windows 内部服务异常）
# 0xFE = 禁用所有 IPv6 接口但保留回环地址
Set-ItemProperty -Path 'HKLM:\SYSTEM\CurrentControlSet\Services\Tcpip6\Parameters' -Name 'DisabledComponents' -Value 0xFE -Type DWord

# 禁用 DNS 多播（减少局域网广播）
New-Item -Path 'HKLM:\SOFTWARE\Policies\Microsoft\Windows NT\DNSClient' -Force | Out-Null
Set-ItemProperty -Path 'HKLM:\SOFTWARE\Policies\Microsoft\Windows NT\DNSClient' -Name 'EnableMulticast' -Value 0 -Type DWord
```

### 9.2 TCP 网络优化

```powershell
& netsh int tcp set global autotuninglevel=normal
& netsh int tcp set global rss=enabled
& netsh int tcp set global timestamps=disabled
```

> ℹ️ `chimney` 选项已在 Windows Server 2012 R2 之后移除，Server 2022 不支持，**不要执行** `chimney=disabled`，否则报错。

## 第11步：修复 RDP 会话 DWM 崩溃问题（Azure VM 专用）

> 如果执行优化后 RDP 连接时报错 **"Your session was logged off because DWM crashed"**，需执行此步。

**原因：** Azure VM 使用 `Microsoft Basic Display Adapter`（软件渲染）作为虚拟显示适配器，该适配器不支持完整的硬件 GPU 加速。DWM（桌面窗口管理器）启动时若尝试初始化硬件加速，则会崩溃并断开 RDP 会话。

**修复方法：** 禁用 DWM 硬件加速，强制使用软件渲染：

```powershell
# 禁用DWM硬件加速（防止Azure Basic Display Adapter导致DWM崩溃）
if (-not (Test-Path 'HKCU:\Software\Microsoft\Windows\DWM')) {
    New-Item -Path 'HKCU:\Software\Microsoft\Windows\DWM' -Force | Out-Null
}
Set-ItemProperty -Path 'HKCU:\Software\Microsoft\Windows\DWM' -Name 'DisableHWAcceleration' -Value 1 -Type DWord
Write-Host "DWM硬件加速已禁用，重启后RDP不再崩溃"
```

**同时确认 DWM 关键依赖服务正常运行：**

```powershell
# 确保 UxSms（DWM会话管理器）和 Themes 服务为自动启动
foreach ($svc in @('UxSms', 'Themes')) {
    Set-Service -Name $svc -StartupType Automatic -ErrorAction SilentlyContinue
    Start-Service -Name $svc -ErrorAction SilentlyContinue
    Write-Host "已确认: $svc → 自动启动"
}
```

> ✅ 设置后**重启系统**，再次 RDP 连接将不再出现 DWM 崩溃错误。
